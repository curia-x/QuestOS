## Notes For Maintainers

- Keep upstream copyright notices and SPDX headers intact.
- When refreshing a vendored subtree, record the upstream URL, tag or commit,
  import date, and any local modifications in this file.
- If generated files are committed, record the generator version and source
  configuration where practical.
- Before a public release, verify that every redistributed file has a clear
  license header or is covered by an entry in this notice file.
